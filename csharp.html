Introdução Variáveis e Tipos Estruturas de Controlo Métodos Classes Exercícios Soluções

1. Introdução

C# é uma linguagem orientada a objetos, usada tanto para aplicações desktop, web (com .NET), jogos (Unity), entre outros. Para correr exemplos localmente, instale o .NET SDK (versão recomendada 6 ou 7) e usa o comando "dotnet new console" seguido de "dotnet run".

Exemplo: programa "Hello World" 
using System;
class Program 
{
    static void Main() 
    {
        Console.WriteLine("Hello World!");
    }
}

2. Variáveis e Tipos

Tipos primitivos principais em C#:

  • int — número inteiro (ex.: 0, 1, -10)
  • double — número com casas decimais (ex.: 3.14)
  • string — texto
  • char — caracter (ex.: 'a')
  • bool — booleano (true ou false)

Declaração e atribuição:

int idade = 17;
double preco = 4.99;
string nome = "Leonor";
bool ativo = true;
char letra = 'L';

Dica: Utilize nomes significativos e siga as convenções (camelCase para variáveis locais).

3. Estruturas de Controlo

3.1 Condicionais

int nota = 12;
if (nota >= 16) 
{
    Console.WriteLine("Excelente");
} else if (nota >= 10 && nota < 16) {
    Console.WriteLine("Suficiente");
} else {
    Console.WriteLine("Insuficiente");
}

3.2 Loops

For, while, e foreach:

// for
for (int i = 0; i < 5; i++) 
{
    Console.WriteLine(i);
}

// while
int j = 0;
while (j < 5) {
    Console.WriteLine(j);
    j++;
}

// foreach
    string[] nomes = { "Ana", "Bruno", "Alexandre" };
    foreach (var n in nomes) 
    {
        Console.WriteLine(n);
    }

Explicação: For é utilizado quando conheces o número de iterações; while quando depende de uma condição; foreach para percorrer arrays/coleções.

4. Métodos

Métodos (funções) ajudam a organizar código e a evitar repetição.

static int Soma(int a, int b) 
{
    return a + b;
}
static void Main() 
{
    int resultado = Soma(3, 4);
    Console.WriteLine(resultado); // 7
}

5. Classes

C# usa classes para modelar objetos - Construtores e Propriedades get/set:

class Pessoa
{
    private string nome;
    private int idade;

    public Pessoa(string nome, int idade) 
    {
        this.nome = nome;
        this.idade = idade;
    }

    public string GetNome() { return nome; }
    public void SetNome(string n) { nome = n; }

    public int GetIdade() { return idade; }
    public void SetIdade(int i) { idade = i; }
}
            
static void Main() 
{
    Pessoa p = new Pessoa("Leonor", 17);
    Console.WriteLine(p.GetNome());
}

Exercícios

Exercício 1

Cria um programa que peça ao utilizador o nome e a idade e mostre uma mensagem: "Olá <nome>, tens <idade> anos."

Exercício 2

Faz um programa que calcule a soma dos números de 1 a N (N fornecido pelo utilizador) utilizando um loop.

Exercício 3

Cria uma classe ContaBancaria com saldo (privado), métodos Depositar, Retirar e MostrarSaldo. Testa a classe no programa principal.

Soluções (exemplos)

Exercício 1 — solução exemplo

using System;
class Program 
{
    static void Main() 
    {
        Console.Write("Nome: ");
        string nome = Console.ReadLine();
        Console.Write("Idade: ");
        int idade = int.Parse(Console.ReadLine());
        Console.WriteLine($"Olá {nome}, tens {idade} anos.");
    }
}

Exercício 2 — solução exemplo

using System;
class Program 
{
    static void Main() 
    {
        Console.Write("N: ");
        int N = int.Parse(Console.ReadLine());
        int soma = 0;
        for (int i = 1; i <= N; i++) 
        {
            soma += i;
        }
        Console.WriteLine($"Soma 1..{N} = {soma}");
    }
}

Exercício 3 — solução exemplo

using System;
class ContaBancaria 
{
    private double saldo;

    public ContaBancaria(double inicial) 
    {
    saldo = inicial;
    }

    public void Depositar(double valor) 
    {
        if (valor > 0) saldo += valor;
    }

    public bool Retirar(double valor) 
    {
        if (valor > 0 && valor <= saldo) 
        {
            saldo -= valor;
            return true;
        }
        return false;
    }

    public double MostrarSaldo() 
    {
        return saldo;
    }
}
                        
class Program 
{
    static void Main() 
    {
        ContaBancaria c = new ContaBancaria(100);
        c.Depositar(50);
        bool ok = c.Retirar(30);
        Console.WriteLine($"Retiro efectuado? {ok}. Saldo = {c.MostrarSaldo()}");
    }
}
